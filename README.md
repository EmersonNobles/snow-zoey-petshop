# Snow e Zoey Petshop

Projeto acadêmico desenvolvido para a disciplina de **Análise e Desenvolvimento de Sistemas (ADS)** na PUC.

## Fase 2: Bootstrap, JavaScript e Acessibilidade

Nesta fase, foi adicionada estilização com Bootstrap 5.3, CSS personalizado, JavaScript com funções dinâmicas e acessibilidade para deficientes visuais.

## Páginas do sistema

| Página | Descrição |
|--------|-----------|
| index.html | Página inicial com carrossel de imagens, saudação dinâmica e cards |
| caes.html | Produtos para Cães (2 produtos com cards Bootstrap) |
| gatos.html | Produtos para Gatos (2 produtos com cards Bootstrap) |
| aves.html | Produtos para Aves (2 produtos com cards Bootstrap) |
| servicos.html | Serviços do petshop (banho/tosa, adestramento, creche) |
| cadastro.html | Formulário de cadastro do cliente e do pet |
| agendamento.html | Agendamento de banho e tosa com seleção de data e horário |
| ajuda.html | Documentação completa do sistema |

## Funcionalidades JavaScript

- Relógio em tempo real exibido na navbar
- Saudação dinâmica por período do dia (bom dia / boa tarde / boa noite)
- Contador de visitas com localStorage
- Validação do formulário de cadastro com mensagens de erro
- Seleção visual de serviço no agendamento
- Validação do formulário de agendamento

## Tecnologias utilizadas

- HTML5 semântico
- CSS3 com variáveis customizadas (`css/style.css`)
- Bootstrap 5.3 — navbar responsiva, carrossel, grid, cards, formulários
- Bootstrap Icons
- JavaScript puro (ES6) — `js/main.js`
