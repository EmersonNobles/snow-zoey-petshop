# Snow e Zoey Petshop

Projeto acadêmico desenvolvido para a disciplina de **Análise e Desenvolvimento de Sistemas (ADS)** na PUC.

## Fase 1: estrutura HTML

Nesta fase, foi construída a estrutura base do site utilizando **HTML5 puro**, sem CSS ou JavaScript.

## Páginas do sistema

| Página | Descrição |
|--------|-----------|
| index.html | Página inicial com boas-vindas e visão geral |
| caes.html | Produtos para Cães (2 produtos) |
| gatos.html | Produtos para Gatos (2 produtos) |
| aves.html | Produtos para Aves (2 produtos) |
| servicos.html | Serviços do petshop (banho/tosa, adestramento, creche) |
| ajuda.html | Documentação e ajuda do sistema |

## Tecnologias utilizadas

- HTML5 semântico.
